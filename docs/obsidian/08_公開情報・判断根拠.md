# 08_公開情報・判断根拠

## 判断の要約

本プロジェクトの方向性は、公開ドキュメントで確認できるSnowflake/dbt/AI Agent周辺の技術テーマに沿っている。ただし、特定の組織や案件の標準構成を再現しているとは断定しない。

公開情報から強く支持されるのは、以下。

- Snowflake / dbt を中心にしたデータ基盤モダナイゼーション
- Snowflake Intelligence / Cortex Analyst / Cortex Search / Cortex Agents への関心
- 流通・小売における売上、在庫、週報、BI、異常検知、需要予測との接点
- 生成AI導入における評価、ガードレール、運用、セキュリティ、コスト最適化
- AI Agent による外部ツール/業務システム連携

## 選定した推論

- 小売・流通ドメインは公開事例とAI Agentページの両方に接点があるため題材にする
- Snowflake-native first は構造化データとAIの境界を学ぶ題材として価値が高い
- LangGraph/OpenAI Agents SDK は主役ではなく、agent internals理解の比較対象にする
- preprocessing/security/cost は、AI側の境界設計として扱う

## 断定しないこと

- 特定の組織が全案件でCortex Agentsを使う
- 特定の組織が全案件でLangGraphを使う
- RAG chunking/metadata設計だけが主戦場である
- 小売以外より小売だけが多い
- このリポジトリが特定組織の公式構成を再現している
